# 🐍 Classic Snake Game

Moja własna implementacja kultowej gry **Snake** stworzona w języku Python. Projekt skupia się na czystym kodzie, programowaniu obiektowym (OOP) oraz funkcjonalnościach, które sprawiają, że gra jest "kompletna".

---

## ✨ Kluczowe funkcje

* **System rekordów (High Score):** Gra automatycznie odczytuje i zapisuje Twój najlepszy wynik w zewnętrznym pliku `data.txt`. Rekord zostaje z Tobą nawet po zamknięciu programu!
* **Obramowanie planszy:** Dynamicznie rysowana niebieska ramka wyznaczająca bezpieczny obszar gry.
* **Płynna animacja:** Dzięki użyciu metody `tracer(0)`, gra działa płynnie i bez zbędnego mrugania elementów.
* **Logika kolizji:** System wykrywa uderzenia w ściany oraz we własny ogon węża.

## 🛠️ Technologie

* **Python 3**
* **Turtle Graphics Library** (grafika i interakcja)
* **File I/O** (trwałe przechowywanie danych)
